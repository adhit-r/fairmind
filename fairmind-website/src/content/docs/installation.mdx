# Installation Guide

This guide provides detailed installation instructions for the Fairmind AI Governance Platform across different environments and deployment options.

## System Requirements

### Minimum Requirements
- **Operating System**: macOS 10.15+, Ubuntu 18.04+, Windows 10+
- **Python**: 3.9 or higher
- **Node.js**: 18.0 or higher
- **Memory**: 8GB RAM minimum (16GB recommended)
- **Storage**: 10GB free space
- **Network**: Internet connection for package downloads

### Recommended Requirements
- **Operating System**: macOS 12+, Ubuntu 20.04+, Windows 11+
- **Python**: 3.11 or higher
- **Node.js**: 20.0 or higher
- **Memory**: 16GB RAM or higher
- **Storage**: 50GB free space (SSD recommended)
- **GPU**: NVIDIA GPU with CUDA support (optional, for ML acceleration)

## Quick Installation

### Using the Installer Script

```bash
# Download and run the installer
curl -fsSL https://install.fairmind.xyz | bash

# Or using wget
wget -qO- https://install.fairmind.xyz | bash
```

### Manual Installation

#### 1. Clone the Repository

```bash
git clone https://github.com/your-org/fairmind-ethical-sandbox.git
cd fairmind-ethical-sandbox
```

#### 2. Install Python Dependencies

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# On macOS/Linux:
source venv/bin/activate
# On Windows:
venv\Scripts\activate

# Install Python packages
pip install -r backend/requirements.txt
```

#### 3. Install Node.js Dependencies

```bash
# Install bun (recommended package manager)
curl -fsSL https://bun.sh/install | bash

# Install dependencies
bun install
```

#### 4. Configure Environment

```bash
# Copy environment template
cp .env.example .env

# Edit environment variables
nano .env
```

Required environment variables:

```env
# Database Configuration
DATABASE_URL=postgresql://username:password@localhost:5432/fairmind
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_anon_key

# API Keys
OPENAI_API_KEY=your_openai_api_key
ANTHROPIC_API_KEY=your_anthropic_api_key

# Security
SECRET_KEY=your_secret_key
JWT_SECRET=your_jwt_secret

# Monitoring
TINYBIRD_TOKEN=your_tinybird_token
```

#### 5. Initialize Database

```bash
# Run database migrations
cd backend
alembic upgrade head

# Seed initial data
python scripts/seed_data.py
```

#### 6. Start the Application

```bash
# Start all services
bun run dev

# Or start individually:
# Backend API
cd backend && python main.py

# Frontend
bun run dev:frontend

# Documentation
bun run dev:docs
```

## Platform-Specific Instructions

### macOS Installation

#### Using Homebrew

```bash
# Install Homebrew if not installed
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install dependencies
brew install python@3.11 node@20 postgresql@15

# Install Fairmind
brew install fairmind/tap/fairmind
```

#### Using MacPorts

```bash
# Install MacPorts dependencies
sudo port install python311 nodejs20 postgresql15

# Follow manual installation steps above
```

### Ubuntu/Debian Installation

#### Using APT

```bash
# Update package list
sudo apt update

# Install system dependencies
sudo apt install -y python3.11 python3.11-venv nodejs npm postgresql postgresql-contrib

# Install Fairmind
curl -fsSL https://install.fairmind.xyz | sudo bash
```

#### Using Snap

```bash
# Install via Snap
sudo snap install fairmind --classic
```

### Windows Installation

#### Using Chocolatey

```powershell
# Install Chocolatey if not installed
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# Install dependencies
choco install python nodejs postgresql

# Install Fairmind
choco install fairmind
```

#### Using Winget

```powershell
# Install dependencies
winget install Python.Python.3.11
winget install OpenJS.NodeJS
winget install PostgreSQL.PostgreSQL

# Follow manual installation steps
```

## Docker Installation

### Using Docker Compose

```bash
# Clone repository
git clone https://github.com/your-org/fairmind-ethical-sandbox.git
cd fairmind-ethical-sandbox

# Start all services
docker-compose up -d

# Check status
docker-compose ps
```

### Docker Compose Configuration

```yaml
version: '3.8'
services:
  frontend:
    build: ./frontend
    ports:
      - "3000:3000"
    environment:
      - NEXT_PUBLIC_API_URL=http://backend:8000
    depends_on:
      - backend

  backend:
    build: ./backend
    ports:
      - "8000:8000"
    environment:
      - DATABASE_URL=postgresql://postgres:password@db:5432/fairmind
    depends_on:
      - db

  db:
    image: postgres:15
    environment:
      - POSTGRES_DB=fairmind
      - POSTGRES_USER=postgres
      - POSTGRES_PASSWORD=password
    volumes:
      - postgres_data:/var/lib/postgresql/data

  docs:
    build: ./fairmind-website
    ports:
      - "4321:4321"

volumes:
  postgres_data:
```

## Cloud Deployment

### AWS Deployment

#### Using AWS ECS

```bash
# Install AWS CLI
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install

# Configure AWS credentials
aws configure

# Deploy to ECS
aws ecs create-cluster --cluster-name fairmind-cluster
aws ecs register-task-definition --cli-input-json file://task-definition.json
aws ecs create-service --cluster fairmind-cluster --service-name fairmind-service --task-definition fairmind:1
```

#### Using AWS Lambda

```bash
# Package application
zip -r fairmind-lambda.zip . -x "node_modules/*" "venv/*"

# Deploy to Lambda
aws lambda create-function \
  --function-name fairmind-api \
  --runtime python3.11 \
  --handler main.handler \
  --zip-file fileb://fairmind-lambda.zip
```

### Google Cloud Deployment

#### Using Google Cloud Run

```bash
# Install Google Cloud SDK
curl https://sdk.cloud.google.com | bash
exec -l $SHELL

# Configure project
gcloud config set project your-project-id

# Deploy to Cloud Run
gcloud run deploy fairmind \
  --source . \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated
```

### Azure Deployment

#### Using Azure Container Instances

```bash
# Install Azure CLI
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

# Login to Azure
az login

# Deploy to Container Instances
az container create \
  --resource-group myResourceGroup \
  --name fairmind \
  --image fairmind/fairmind:latest \
  --dns-name-label fairmind-dns \
  --ports 3000 8000
```

## Troubleshooting

### Common Issues

#### Python Version Issues

```bash
# Check Python version
python --version

# If version is incorrect, use pyenv
pyenv install 3.11.0
pyenv global 3.11.0
```

#### Node.js Version Issues

```bash
# Check Node.js version
node --version

# If version is incorrect, use nvm
nvm install 20
nvm use 20
```

#### Database Connection Issues

```bash
# Check PostgreSQL status
sudo systemctl status postgresql

# Start PostgreSQL if not running
sudo systemctl start postgresql

# Create database and user
sudo -u postgres psql
CREATE DATABASE fairmind;
CREATE USER fairmind_user WITH PASSWORD 'password';
GRANT ALL PRIVILEGES ON DATABASE fairmind TO fairmind_user;
```

#### Port Conflicts

```bash
# Check what's using port 3000
lsof -i :3000

# Kill process if needed
kill -9 <PID>
```

### Performance Optimization

#### Memory Optimization

```bash
# Increase Node.js memory limit
export NODE_OPTIONS="--max-old-space-size=4096"

# Optimize Python memory usage
export PYTHONOPTIMIZE=1
```

#### Database Optimization

```sql
-- Optimize PostgreSQL settings
ALTER SYSTEM SET shared_buffers = '256MB';
ALTER SYSTEM SET effective_cache_size = '1GB';
ALTER SYSTEM SET maintenance_work_mem = '64MB';
ALTER SYSTEM SET checkpoint_completion_target = 0.9;
ALTER SYSTEM SET wal_buffers = '16MB';
ALTER SYSTEM SET default_statistics_target = 100;
```
