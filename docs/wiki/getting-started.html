Getting Started

Install and configure FairMind for AI governance

Prerequisites

Node.js 18+

Required for running the frontend and development tools

node --version

Bun Package Manager

Fast JavaScript runtime and package manager

curl -fsSL https://bun.sh/install | bash

Python 3.9+

Required for backend services and ML components

python --version

Git

Version control for collaborative development

git --version

Installation Guide

1

Clone the Repository

Start by cloning the FairMind repository to your local machine

git clone https://github.com/fairmind/ethical-sandbox
cd fairmind-ethical-sandbox
2

Install Dependencies

Install all required dependencies for both frontend and backend

Frontend Dependencies

bun install

Backend Dependencies

cd apps/backend
uv sync
3

Configure Environment

Set up environment variables and configuration files

cp .env.example .env
# Edit .env with your configuration

Important Configuration

  • • Set API keys for external services
  • • Configure database connections
  • • Set up authentication providers
  • • Configure monitoring endpoints
4

Start Services

Launch the development servers for both frontend and backend

# Start all services
bun run dev

# Or start individually
bun run dev:frontend
bun run dev:backend

Frontend

http://localhost:3000

Backend API

http://localhost:8001

Verification

Check Installation

Verify that all services are running correctly by checking the following endpoints

Frontend Dashboard

http://localhost:3000

Backend API Health

http://localhost:8001/health

Governance API

http://localhost:8001/api/v1/governance/metrics
