Governance Endpoints

Bias Detection Endpoints

POST

Analyze Bias

Perform comprehensive bias analysis on AI models

POST /api/v1/bias/analyze

Request Parameters

Parameter Type Required Description
modelId string Required ID of the model to analyze
datasetName string Optional Name of the dataset to use
sensitiveColumns string[] Optional List of sensitive columns to check

Request Example

{ "modelId": "model_123", "datasetName": "credit_data", "sensitiveColumns": ["gender", "race", "age"] }

Response Example

{
  "id": "analysis_456",
  "modelId": "model_123",
  "status": "completed",
  "fairnessScore": 85.2,
  "biasScore": 14.8,
  "biasTypes": [
    "statistical_parity",
    "equalized_odds"
  ],
  "recommendations": [
    "Consider feature engineering to reduce bias",
    "Implement post-processing techniques"
  ]
}

Security Testing Endpoints

POST

Run Security Analysis

Perform comprehensive security testing using OWASP guidelines

POST /api/v1/security/analyze

Request Parameters

Parameter Type Required Description
modelId string Required ID of the model to test
testMode string Optional comprehensive or targeted
testCategories string[] Optional Specific test categories to run

Response Example

{
  "id": "sec_789",
  "modelId": "model_123",
  "overallScore": 92.5,
  "securityScore": 94.2,
  "riskLevel": "low",
  "vulnerabilities": 2,
  "testSummary": {
    "totalTests": 45,
    "passedTests": 43,
    "failedTests": 2
  }
}

Error Codes

Code Message Description
400 Bad Request Invalid request parameters or malformed JSON
401 Unauthorized Missing or invalid API key
403 Forbidden Insufficient permissions for the requested resource
404 Not Found Requested resource does not exist
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Unexpected server error
